# Porchetto ADC
A simple project to read 7 analog signals and printing their digital values into UART.

## Pin used
We are using ADC2, in particular starting from channel IN6 to channels IN12 which corresponds to:
- IN6 $\rightarrow$ PA6
- IN7 $\rightarrow$ PA7
- IN8 $\rightarrow$ PB0
- IN9 $\rightarrow$ PB1
- IN10 $\rightarrow$ PC0
- IN11 $\rightarrow$ PC1
- IN12 $\rightarrow$ PC2
